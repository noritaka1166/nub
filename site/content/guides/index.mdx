Step-by-step guides for integrating Nub into an existing project or migrating from another tool.

## Available guides

- [Migrating from Bun to Nub](/guides/bun-to-nub) — move a Bun project onto Node + Nub: runtime, scripts, env files, and a complete map of every `Bun.*` API to its Node or npm-ecosystem replacement.
- [Using Nub with Turborepo](/guides/turborepo) — run Turborepo with Nub as the runtime and package manager underneath, keeping your existing `packageManager` field and lockfile.
